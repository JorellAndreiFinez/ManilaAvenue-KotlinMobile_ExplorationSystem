## ğŸ§© Project Rationale

Exploring Manila can be overwhelming for both locals and tourists due to the abundance of options and scattered information. Travelers often spend significant time searching for restaurants, museums, markets, and hidden gems, while locals may struggle to discover new experiences or share recommendations.

**ManilaAvenue** addresses these challenges by providing a centralized, user-friendly Kotlin-based Android application that streamlines city exploration. It allows users to browse nearby places, view detailed information, follow curated recommendations, filter by categories, and interact with the community through posts and profiles.

By integrating programming concepts such as Android development, API integration, Firebase for data persistence, and object-oriented design with a real-world application, this project demonstrates how mobile technologies can enhance user experiences, making discovering Manila seamless, engaging, and personalized. smooth rental experience for travelers.

## ğŸ¯ Project Objectives

### General Objective

The objective of Manila Avenue is to provide users with a seamless and intuitive platform to explore the rich offerings of Manila in real time. By utilizing real-time location tracking, our application allows users to discover nearby restaurants, hidden gems, museums, and markets effortlessly. With a user-friendly interface, Manila Avenue aims to enhance the city exploration experience, making it easy for locals and visitors alike to find and enjoy the best that Manila has to offer, anytime and anywhere.

### Specific Objectives

1. To develop a Kotlin-based Android application that serves as a centralized platform for users to explore, discover, and interact with the best places in Manila.
2. To create a user-friendly mobile interface that allows users to browse categories such as restaurants, parks, museums, markets, and hidden gems, with detailed information, ratings, and location mapping for each place.
3. To demonstrate the implementation of essential mobile development concepts, including API integration, Firebase for authentication and data persistence, object-oriented programming, and interactive UI/UX design, applied to a real-world city exploration system.
